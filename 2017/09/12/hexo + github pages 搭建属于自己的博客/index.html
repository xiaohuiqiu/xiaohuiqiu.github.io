hexo + github pages 搭建属于自己的博客

发表于 2017-09-12

搭建之前需要有：

  1. 配置好ssh的github账号。
  2. 安装node.js。
  3. 安装git。

首先注册github账号，然后打开Settings -> SSH and GPG keys。
其次在终端输入如下命令行 生成本地 SSH keys)。
ssh-keygen -t rsa -C "github邮箱" -t表示要创建的密匙类型 -C表示注释
前往目录~/.ssh/以.pub结尾的就是公钥。用vim打开公钥或者复制一份到桌面，把后缀改为txt直接双击打开复制里面的所有内容粘贴到我们刚才打开的github位置。

node.js 可以去官网下载。或者通过 homebrew 安装

git Mac 自带 不需要额外安装。

安装Hexo

终端执行

然后初始化
先新建一个目录用来管理你的hexo项目，cd到你新建的目录执行

npm安装

检查是否安装成功 开启hexo服务器 server

然后打开http://localhost:4000查看是否正常

关联github

  1. 在你的github新建一个仓库名为用户名.github.io 要固定是这个格式的。
  2. 在你的hexo项目目录找到_config.yml配置文件

打开后滑倒最底部，把要关联的github仓库地址填上

注意填写的时候所有冒号:后面都要跟一个空格，不然会报语法错误

然后退出保存:wq

执行生成静态页面的命令 generate

部署命令 deploy

如果提示无法连接git 需要执行npm install hexo-developer-git –save 命令

再次执行hexo g 和 hexo d命令

这时 如果打开http://用户名.github.io 和 http://localhost:4000 一样说明关联成功了。

发布博客

在自己本地的hexo项目目录 执行

名为博客标题.md的文件会建在目录/source/_posts下。你当然可以用vim来编辑文章。你可以使用markdown编辑器来编辑你的博客。

修改配置

安装主题 主题都放在／themes/目录下。你可以cd到themes 然后通过git下载主题

比如next主题 git clone https://github.com/iissnan/hexo-theme-next

然后修改_config.yml 配置文件里面的theme 为next。
再次执行hexo clean，hexo g 和 hexo d命令。主题就生效了。

修改语言
修改_config.yml 配置文件里面的language 为相应语言。

English
中文简体 (zh-Hans)
French (fr-FR)
正体中文 (zh-hk/zh-tw)
Russian (ru)
German (de)
